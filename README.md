**HighlighterTXT** is a specialized offline desktop environment built for security researchers, sysadmins, and power users. It transforms raw text data into an organized, annotated, and highly secure workspace.

---

### 🚀 Core Engineering

#### 🟢 Standard Engine (Optimization & Analysis)
* **Real-time Annotations:** Instant multi-color highlighting engine for log analysis and pattern discovery.
* **Metadata Layering:** Attach emoji-tags and persistent notes to specific lines without altering the raw source.
* **Navigation Dock:** Context-aware summary panel for rapid jumping between critical data points.
* **PDF Engine:** Automated conversion of annotated workspaces into formatted, professional PDF reports.

#### 🛡️ PRO Security Suite (Hardened Protection)
* **Proprietary `.hxt` Format:** Military-grade AES-256-GCM encryption to secure sensitive credentials and private logs.
* **Workspace Kill-Switch:** Instant UI locking (Ctrl+Alt+L) with master password verification.
* **Idle-AutoLock:** Automated session termination based on configurable inactivity timeouts.
* **Zero-Knowledge Architecture:** 100% offline verification. No cloud, no tracking, total privacy.
